Переваги Flexbox

Flexbox - це потужний інструмент для створення адаптивних макетів. Він дозволяє легко вирівнювати елементи, розподіляти вільний простір і змінювати порядок відображення без зміни HTML.

У цьому прикладі ми використовуємо Flexbox для створення повноцінного макету сайту з шапкою, основним контентом, сайдбаром і підвалом.

Основні властивості Flexbox

Display: flex - перетворює елемент на flex-контейнер.

Flex-direction - встановлює напрямок головної осі (row, column).

Justify-content - вирівнює елементи вздовж головної осі.

Align-items - вирівнює елементи вздовж поперечної осі.

Flex - комбінована властивість для flex-grow, flex-shrink і flex-basis.
